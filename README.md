# 🔢 Super Contador

Este é um projeto interativo desenvolvido para praticar estruturas de repetição em JavaScript. O utilizador define um início, um fim e o "passo" (intervalo), e o sistema realiza a contagem automaticamente com emojis.
![Demonstração do Projeto](Captura%20de%20tela%202026-04-22%20192948.png)



## 🚀 Tecnologias Utilizadas
* **HTML5** (Estrutura)
* **CSS3** (Estilização e Design Responsivo)
* **JavaScript** (Lógica de Contagem e Laços de Repetição)

## 📚 Aprendizados
Neste exercício do curso *Curso em Vídeo*, pratiquei:
- Manipulação de dados de formulários (`input type="number"`)
- Estrutura de repetição `for`
- Validação de dados (verificação de campos vazios ou passo 0)
- Criação dinâmica de conteúdo no HTML via JS
